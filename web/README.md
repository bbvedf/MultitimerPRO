# MultiTimer PRO Landing Page

Official promotional website for MultiTimer PRO.

## Features
- **Multilingual Support**: English, Spanish, French, German, Italian, and Portuguese.
- **Adaptive Themes**: High-contrast Dark Mode and refined Light Mode.
- **Responsive Design**: Optimized for desktop and mobile.
- **Interactive Elements**: Custom language selector, smooth scroll, and tech-inspired animations.

## Local Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/`: React components and logic.
- `public/`: Static assets (logos, screenshots).
- `index.html`: Entry point.
- `vite.config.ts`: Build configuration.
